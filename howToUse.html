Play Page (play.html)

This page allows users to interact with a Sudoku puzzle. Here, you can solve, check, submit, reset, wipe the grid or get a hint! Below are the buttons available on this page with their more detailed explanations:

Button Description
Check Checks if the current state of the Sudoku puzzle is correctly solved. In case it is, this button will turn green, and in case it isn't, it will turn red.
Submit This button allows you to make your own grid! You can add on the already existing grid or you can use the button WIPE to remove any pre-existing numbers and write your own! When you're done creating your grid, click this button and get to solving! This button will automatically run through the code of the check() button to make sure no numbers are clashing.
Hint Provides a hint by filling in one empty cell with a correct number. If a cell is cell-ected, it will put a number inside that cell. if it isn't, it will put a number in a random empty cell inside the grid.
Solve Automatically solves the Sudoku puzzle
Reset Resets the sudoku puzzle to its initial state, removing all the numbers you put in and the ones the hint button did.
Wipe Clears the entire Sudoku puzzle grid, including the pre-written numbers, enabling you to make your own grid! Make sure to press the SUBMIT button to submit your grid so you can properly solve it!
