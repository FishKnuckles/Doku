Using the Play Page

The Play page is where you can enjoy playing sudoku puzzles. Here's how to use it effectively:

Sudoku Grid

Interacting with Cells: Click on any cell within the Sudoku grid to select it. Once selected, you can input numbers from 1 to 9 using your keyboard, or even 0 if you want to remove a number.

Buttons

Here are the functions of the buttons available on the Play page:

Button Name Description
Check Verifies if your current sudoku grid is correct.
Solve Automatically solves the sudoku puzzle for you.
Reset Resets the sudoku grid to its original state.
Wipe Clears all the cells in the sudoku grid.
Hint Provides a hint by filling in one cell of the grid.
Submit Submits your completed sudoku grid for verification.

You can also press the following button which will take you to the How To Use page. How To Use contains a more detailed explanation of each function.
